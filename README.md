# nje_fraud_detection

## Development tools
The following tools are used for development:

- OS - [Fedora KDE 43](https://fedoraproject.org/kde/)
- IDE - [Zed](https://zed.dev/)
- Language - [Go](https://go.dev/)
- Container management - [Podman](https://podman.io/)
- Data serializer - [Protocol buffer](https://protobuf.dev/) compiler `protoc` and [`protoc-gen-go`](https://pkg.go.dev/google.golang.org/protobuf)
- In-memory data store - [Redis image](https://hub.docker.com/_/redis)
- Event streaming - [Redpanda image](https://hub.docker.com/r/redpandadata/redpanda)
- Data visualization - [Grafana image](https://hub.docker.com/r/grafana/grafana)
- Monitoring - [Prometheus image](https://hub.docker.com/r/prom/prometheus)
